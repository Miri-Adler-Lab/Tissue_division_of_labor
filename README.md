# Universal and Tissue-Specific Archetypes of Tissue-Level Division of Labor

This repository contains the analysis code and computational pipeline used in the study:

**“Tissue-level division of labor is coordinated across organs through universal programs and context-specific adaptations”**
Yonassi et al. (2025)

The associated data and analysis outputs are available on Zenodo:

Yonassi, S., Goldenberg, N., Walfisch, A., Cohen, E., Shaer Tamar, E., & Adler, M. (2025). *Tissue-level division of labor is coordinated across organs through universal programs and context-specific adaptations: Analysis code and data* [Dataset]. Zenodo. https://doi.org/10.5281/zenodo.17792755
