# Universal and Tissue-Specific Archetypes of Tissue-Level Division of Labor

This repository contains the analysis code and computational pipeline used in the study:

**“Tissue-level division of labor is coordinated across organs through universal programs and context-specific adaptations”**

Yonassi et al. (2025)

## Repository structure

Each numbered directory is a step of the analysis pipeline:

```
01_Preprocessing/          # Data preprocessing and sampling
02_Archetype_Identification/  # ParTI archetype identification and polytope plots
03_Tissue_Composition/     # Tissue composition and Gini analyses
04_Intercellular_Crosstalk/   # Ligand-receptor / archetype crosstalk (Figure 4)
05_spatial_ligand_receptor/   # Spatial ligand-receptor analysis
06_Archetype_Robustness/   # Archetype robustness (PCA, Jaccard heatmaps)
07_Cross_Species_Analysis/    # Human-mouse cross-species comparison
```
