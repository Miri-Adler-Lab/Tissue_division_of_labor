# Step 04: Intercellular Crosstalk Analysis

This directory contains the ligand-receptor analysis pipeline for identifying archetype-level intercellular communication networks (Figure 4).

## Overview

This step analyzes ligand-receptor interactions between archetypes to reveal task-specific crosstalk across cell types. The analysis uses a Python package structure with Marimo notebooks for interactive exploration.

## Package Structure

```
04_Intercellular_Crosstalk/
├── pyproject.toml          # Python package configuration and dependencies
├── uv.lock                 # Locked dependency versions (managed by uv)
├── src/                    # Source code package
│   └── lr_archetype/       # Main package module
│       ├── __init__.py
│       ├── interactions.py # Core ligand-receptor interaction analysis
│       └── triangle.py     # Archetype geometry utilities
├── code/                # Data preprocessing scripts
│   ├── archetype_enrichment_preproccess.py
│   ├── lr_archetype_interactions.py
│   └── tissue_correlation.py
├── tests/                  # Unit tests
│   ├── test_analysis.py
│   └── test_triangle.py
├── all_interactions.py     # Marimo notebook for interaction network analysis
└── pathways_analysis.py    # Marimo notebook for pathway enrichment analysis
```

## Installation

This step uses a dedicated Python package with dependency management via `uv` (or `pip`).

### Using uv (recommended):
```bash
cd Scripts/04_Intercellular_Crosstalk
uv sync
```

### Using pip:
```bash
cd Scripts/04_Intercellular_Crosstalk
pip install -e .
```

## Required Dependencies

See `pyproject.toml` for the complete list. Key packages include:
- `marimo[lsp]` - Interactive notebook environment
- `scanpy` - Single-cell analysis
- `pandas`, `numpy` - Data manipulation
- `gseapy`, `goatools` - Pathway enrichment
- `altair`, `matplotlib`, `seaborn` - Visualization

## Running the Analysis

### 1. Preprocessing (code/)
First, run the preprocessing scripts to prepare the data:
```bash
python code/archetype_enrichment_preproccess.py
python code/lr_archetype_interactions.py
python code/tissue_correlation.py
```

### 2. Interactive Analysis (Marimo notebooks)
Run the Marimo notebooks for interactive exploration:
```bash
marimo edit all_interactions.py
marimo edit pathways_analysis.py
```

Alternatively, run as scripts:
```bash
marimo run all_interactions.py
marimo run pathways_analysis.py
```

## Input Data

**Required files** (see `MISSING_FILES.md` in repository root):
- `Data/Figure_4_Crosstalk/archetype_lr_pairs.csv`
- `Data/Figure_4_Crosstalk/archetypes_tissue_correlation.csv`

These files should be generated by running the preprocessing scripts with the archetype data from Step 02.

## Output

Results are saved to `Results/Figure_4_Crosstalk/` and include:
- Archetype crosstalk network visualizations
- Process-specific communication networks (angiogenesis, inflammation, ECM, growth factors)
- Pathway enrichment results

Run the test suite to verify the package:

pytest tests/
